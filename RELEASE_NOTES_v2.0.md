# Release Notes - v2.0

## 🎉 重大更新

**论文组图工具 v2.0** 带来了期待已久的功能和重要的稳定性改进！

## ✨ What's New

### 🖱️ 快捷自定义布局
终于可以完全用鼠标定义布局了！
- 规则网格：点击设置行数×列数
- 不规则布局：每行独立设置（自动居中）
- 内置帮助：❓按钮查看详细示例

### 📏 子图宽高比调整
一键统一所有子图比例！
- 7种预设（1:1, 4:3, 16:9等）
- 自定义比例
- 画布高度自动优化

### 📜 滚动面板
参数太多？没问题！
- 右侧面板支持滚动
- 再多参数也不遮挡

## 🐛 Bug Fixes

- 🔥 **修复导入图片时布局混乱** - 现在稳定多了
- 🔥 **修复画布不显示图片** - 画布和预览完全同步
- 🔥 **修复预览不实时更新** - 所有修改立即反映
- 🛠️ 修复初始化错误信息

## 🎨 Improvements

- 不规则布局框尺寸完全统一
- 自动优化行间距
- UI布局优化（更大的预览和参数面板）
- 自定义模式下智能禁用宽高比
